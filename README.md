# Najat 🚑

A crisis-response platform focused on war-related scenarios, providing real-time hospital data, maps, and emergency tracking.

---

## ⚙️ Tech Stack

### 🧭 Routing & Navigation

- Next.js (App Router)

### 📡 Data Fetching

- TanStack Query v5
- tRPC

### 🧠 State Management

- Zustand
- Jotai

### 📝 Forms & Validation

- React Hook Form
- Zod

### 🎨 UI & Styling

- Tailwind CSS v4
- shadcn/ui
- Framer Motion

### ⚡ Performance

- next/image
- Vercel Speed Insights

### 🛠 Developer Experience

- TypeScript (strict mode)
- ESLint (Next.js config)
- Prettier (+ Tailwind plugin)

### 🧪 Testing

- Vitest
- Playwright

---

## 🚀 Features

- 🗺 Interactive hospital map
- 🚨 Real-time emergency updates
- 📍 Location-based services
- 🔄 Live data updates (realtime-ready)
- 🔐 External backend integration

---

## 📦 Getting Started

```bash
npm install
npm run dev
```

---

## 🔗 Backend

This project relies on an external backend (API / tRPC) for handling data and business logic.
